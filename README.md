 # Tint and Shades Generator
 
 ## Short Description
 A focused tool for generating consistent tint and shade color scales from a single base color.
 
 ## Problem Statement
 Designers and developers need predictable tint and shade scales to build accessible, consistent UI systems. Creating these scales manually is time-consuming and often inconsistent across teams and files, which makes accessibility checks and design system maintenance harder.
 
 ## Features
 - HEX color input with live preview
 - Adjustable tint and shade variant sliders
 - Color swatch display with copy-to-clipboard
 - CSS variables output with one-click copy
 - Dark theme and responsive layout
 
 ## Tech Stack
 - React
 - TypeScript
 - Vite
 - Tailwind CSS
 - Framer Motion
 
 ## Installation Instructions
 1. Install dependencies:
    - `npm install`
 2. Start the development server:
    - `npm run dev`
 3. Build for production:
    - `npm run build`
